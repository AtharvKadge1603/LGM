## If you don't see changes on GitHub
Your local commits only appear on GitHub after pushing to a configured remote.

```bash
git remote -v
git branch --show-current
git push -u origin <your-branch-name>
```

If `git remote -v` shows nothing, add your GitHub repo first:

```bash
git remote add origin https://github.com/<your-username>/<your-repo>.git
git push -u origin <your-branch-name>
```
